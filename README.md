# Petpat
## Overview
This project draws an idea of a platform with a vision to avail pet care services for users from expert service providers at their doorsteps. Browsing through the application users will be able to order pet care services from the available service providers reviewing their profile details including the cost of the services.

Service providers will get request notifications and will go to the users’ location approving the service requests. Upon completing the service s/he will get a review from the users for his service. 
The system admin, utilizing the admin panel will be able to manage all the administrative features of the platform.

### Definitions & Abbreviations:
* ⭐ Admin-The admin is the controller of the application and is responsible for all tasks and functionalities according to functional requirements.
* ⭐ Service provider-Service provider(s) is an end-user(s) who will be able to register and accept/cancel booking requests for her services. 
* ⭐ User-User(s) is the end-user(s) who will be able to search, browse, register, and book services using the mobile app.
